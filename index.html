What is CSS ?

  1. CSS Stands For Cascading Style sheets

  2. CSS describes how HTML elements are to be displayed on screen,paper or in the media.

  3. CSS is not necessaryCSS saves a lot of work It can control the layout of multiple web pages all at once

  4. External style sheets are stored in CSS files.

CSS can be added to HTML documents in 3 ways:

  • Inline - by using the style attribute inside HTML Elements
  • Internal- by using a style element in the head section
  • External-by using a link element to link to an external CSS file .
  • The most common way to add CSS ,is to keep the styles in external CSS files . However ,this tutorial we will use inline & internal styles, because this is easier to demonstrate ,and easier for you to try it yourself .

When comparing CSS class vs ID ,the difference is that css class applies a styles to multiple elements .ID on the other hand applies a style to one unique element.ID is also special in that you can use a special URL to link directly to an element and its used by java script.

In CSS ,selectors are used to target a specific element or range of elements on a web page.Once an element has been targeted,a style or set of styles can be applied to the element.

There is a wide range of selections available .Two of the most commonly used are class & ID .Both are used to target elements to which a style should be applied.
